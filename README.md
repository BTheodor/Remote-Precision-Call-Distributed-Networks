# Remote-Precision-Call-Distributed-Networks
Remote Procedure Call implementation using tRPC v11 + Next.js 14 + Zod. Educational project mapping core RPC theory (marshalling, stubs, IDL, idempotency) to a working full-stack TypeScript app. Includes a math service (remote_sum, remote_divide, batch operations) and a CRUD todo service with idempotency patterns and structured error handling.
